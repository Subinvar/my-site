# Дополнительные текстовые документы

Краткий список файлов, которые можно добавить позже, если команде понадобится больше регламентов.

- **PUBLISHING-CHECKLIST.md** — пошаговый чек-лист выпуска контента (новости, продукты, документы). Помогает не забыть про локализацию, медиа, ссылки и обновление `SITEMAP`.
- **DEPLOYMENT.md** — быстрый гид по окружениям и деплою (Vercel/CI), с типичным порядком запуска проверок и прогоном e2e перед выкладкой.
- **CONTENT-STYLE.md** — мини-гайд по стилю контента: тон, длина лидов, формат заголовков, правила для таблиц и ссылок. Удобно для редакторов.
- **ACCESSIBILITY.md** — заметки про aria-лейблы, контраст и фокусное кольцо в UI, чтобы новые компоненты соответствовали текущим практикам.
- **ANALYTICS-SEO.md** — кратко о счетчиках, событиях и базовой SEO-гигиене (title/description/og-tags), чтобы не потерять метрики при обновлениях.

> Сейчас эти файлы необязательны: базовая информация уже есть в `README`, `SITEMAP` и `README-UX`. Добавляйте конкретный файл, когда появится процесс, который нужно закрепить текстом.
